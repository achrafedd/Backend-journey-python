# What status code did each request return? What does each mean?
- GET: 200
- POST: 201
- PATCH: 200
- DELETE: 200

# What is the difference between PUT and PATCH?
- We use PATCH method when we need a partial update
- We use PUT method when we need to full replacement

# What does the Content-Type: application/json header tell the server?
- tell's the server that the body is formated as a json
